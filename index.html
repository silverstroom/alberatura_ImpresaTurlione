Specifiche tecniche & SEO

Requisiti tecnici e di indicizzazione che il nuovo sito custom deve rispettare per massimizzare il posizionamento organico e chiudere il gap competitivo con L.T.R. Asphalt.

Architettura URL

  • URL parlanti con keyword + località
  • Trattini (-) come separatori
  • No parametri inutili o /index.php/
  • Profondità max 3 livelli
  • Redirect 301 dal vecchio Joomla

Meta tag obbligatori

  • Title 55-65 caratteri (con località per pagine geo)
  • Meta description 150-160 con CTA
  • H1 unico per pagina
  • Open Graph per social sharing
  • Canonical su tutte le pagine

Schema markup (JSON-LD)

  • LocalBusiness (sede Baragiano)
  • Organization con NAP coerente
  • Product per ogni miscela tecnica
  • FAQPage sulle pagine prodotto
  • BreadcrumbList su tutte le pagine

Performance

  • Core Web Vitals tutti verdi
  • LCP < 2.5s
  • Immagini lazy-load + formato WebP
  • HTTPS obbligatorio (TLS 1.3)
  • Mobile-first responsive

Accessibilità WCAG 2.1 AA

  • Contrasto colori 4.5:1 minimo
  • Alt text descrittivo per ogni immagine
  • Navigazione da tastiera
  • Font ridimensionabili
  • ARIA label sui form di preventivo
